# Intrusion Detection & Prevention Strategy

## NIDS (Network-Based Intrusion Detection)
- Monitors network traffic for malicious activity
- Provides alerts for suspicious behavior
- Cost-effective for large environments

## HIPS (Host-Based Intrusion Prevention)
- Installed on endpoints
- Actively blocks malicious activity
- Provides an additional layer of protection if threats bypass NIDS

## Hybrid Approach
Combining NIDS and HIPS creates a layered defense:
- NIDS detects threats at the network level
- HIPS prevents threats at the host level
